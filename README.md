# 우아한 테크 코스 <br> Java Movie

## 프로젝트 소개
영화를 예매하는 프로그램입니다.  
영화를 예매할 때, 예매가 불가능한 경우인지 확인합니다.  
포인트를 사용할 수 있으며, 결제 유형에 따라 할인율이 달라집니다.  
 
## 기능 목록
- 예약할 영화를 선택합니다.
- 정수인지 확인합니다.
- 영화 목록에 있는 아이디인지 확인합니다.
- 선택한 영화의 시간표를 출력합니다.
- 예약할 시간표를 선택합니다.
- 정수인지 확인합니다.
- 상영 시간이 지났는지 확인합니다.
- 존재하는 시간표인지 확인합니다.
- 예약할 인원을 입력합니다.
- 정수인지 확인합니다.
- 가능한 인원인지 확인합니다.
- 예약할 영화 객체 생성
- 추가 예약을 할 경우, 상영 시간 차이가 1시간 이내인지 확인합니다.
- 예약 내역을 출력합니다.
- 포인트를 입력합니다.
- 결제 방식에 따라 할인을 적용합니다.
- 최종 예매 금액을 출력합니다.
